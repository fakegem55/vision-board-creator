# Vision Board Creator — Requirements

> 서비스 한 줄 요약: 키워드 몇 개와 한 줄 다짐만 입력하면, 나만의 비전보드가 자동으로 완성되는 모바일 웹 서비스

---

## 1. 서비스 개요

| 항목 | 내용 |
|------|------|
| 대상 사용자 | 디지털 기기가 낯선 사용자, 비전보드를 처음 만들어보는 사람 |
| 제공 채널 | 모바일 웹 (반응형, 최대 너비 448px 기준) |
| 핵심 가치 | **단순함** — 입력 2개, 버튼 1개로 완성 |
| 지원 언어 | 한국어 · English · 日本語 · 中文 (기기 언어 자동 감지) |
| 저장/공유 | 기기 로컬 저장 (PNG 다운로드) + OS 공유 시트 (카카오톡 등) |

---

## 2. 사용자 흐름 (Happy Path)

```
① 추천 키워드 선택 또는 직접 입력 (1~9개 태그)
      예: "여행" "건강" "가족" "성공"
② 한 줄 다짐 텍스트 입력 (선택, 최대 60자)
      예: "2026년, 나는 매일 성장한다"
③ "내 비전보드 만들기" 버튼 탭
      → 키워드 자동 번역 (내장 사전 → 캐시 → API)
      → 키워드별 이미지 자동 수집 (Unsplash, 중복 제거)
      → 이미지 + 텍스트 자동 배치
④ 완성된 보드 확인
      → PNG 저장 (카메라롤)
      → 공유 (카카오톡·문자 등 OS 공유 시트)
      → 재생성 (같은 키워드, 다른 이미지 조합)
```

---

## 3. User Stories

### US-01 추천 키워드 선택
> **As a** 무슨 키워드를 입력할지 모르는 사용자,
> **I want to** 카테고리별로 미리 준비된 추천 키워드를 탭해서 바로 추가하고 싶다,
> **So that** 입력 고민 없이 바로 비전보드를 시작할 수 있다.

**Acceptance Criteria:**
- [x] 7개 카테고리(건강, 여행, 가족, 커리어, 재정, 자기계발, 라이프스타일)의 추천 키워드가 칩 형태로 표시된다
- [x] 추천 키워드 탭 시 즉시 키워드 목록에 추가된다
- [x] 이미 선택된 키워드는 강조 표시되고 중복 추가되지 않는다
- [x] 9개 최대치 도달 시 미선택 키워드 칩이 비활성화된다
- [x] 선택 언어에 따라 추천 키워드가 해당 언어로 표시된다

---

### US-02 키워드 직접 입력
> **As a** 추천 목록에 없는 키워드를 넣고 싶은 사용자,
> **I want to** 원하는 키워드를 직접 입력하고 싶다,
> **So that** 나만의 개성 있는 비전보드를 만들 수 있다.

**Acceptance Criteria:**
- [x] 텍스트 입력 후 Enter 또는 스페이스로 키워드 태그가 생성된다
- [x] 태그는 최소 1개, 최대 9개까지 입력 가능하다
- [x] 태그의 × 버튼으로 삭제할 수 있다
- [x] 9개 초과 시 입력이 차단되고 안내 메시지가 표시된다
- [x] 현재 키워드 수 / 최대 수 카운터가 표시된다

---

### US-03 다짐 텍스트 입력
> **As a** 나의 다짐을 보드에 담고 싶은 사용자,
> **I want to** 한 줄 문장을 입력하고 싶다,
> **So that** 이미지와 함께 나의 의지를 시각적으로 표현할 수 있다.

**Acceptance Criteria:**
- [x] 텍스트 입력 필드 1개가 제공된다
- [x] 최대 60자 제한이 있다
- [x] 입력하지 않아도 보드 생성이 가능하다 (선택 사항)
- [x] 다짐 텍스트는 보드 **중앙**에 반투명 배경 + 서체(Dancing Script)로 표시된다
- [x] 앞뒤 큰따옴표(" ")로 다짐이 강조된다

---

### US-04 비전보드 자동 생성
> **As a** 디자인을 모르는 사용자,
> **I want to** 버튼 하나로 보드가 자동으로 만들어지길 원한다,
> **So that** 레이아웃이나 디자인을 직접 고민하지 않아도 된다.

**Acceptance Criteria:**
- [x] "만들기" 버튼 탭 시 키워드별로 Unsplash 이미지를 자동 수집한다
- [x] 키워드는 내장 사전 → localStorage 캐시 → 번역 API 순서로 영어로 변환된다
- [x] 중복 이미지가 배정되지 않는다 (키워드마다 고유 사진)
- [x] 이미지들이 빈 칸 없이 보드에 자동 배치된다
- [x] 다짐 텍스트가 보드 중앙에 오버레이된다
- [x] 로딩 중 스피너가 표시된다

---

### US-05 보드 재생성
> **As a** 결과가 마음에 들지 않는 사용자,
> **I want to** 같은 키워드로 다른 이미지 조합을 시도하고 싶다,
> **So that** 내 취향에 맞는 결과를 찾을 수 있다.

**Acceptance Criteria:**
- [x] "다시 만들기" 버튼 탭 시 같은 키워드로 새 이미지 조합이 생성된다
- [x] 이전과 다른 이미지가 선택된다 (랜덤 페이지 오프셋 적용)
- [x] 랜덤 페이지 결과가 없으면 1페이지로 폴백된다

---

### US-06 PNG 저장
> **As a** 완성한 비전보드를 기기에 저장하고 싶은 사용자,
> **I want to** 보드를 이미지 파일로 다운로드하고 싶다,
> **So that** 배경화면이나 앨범에서 언제든 볼 수 있다.

**Acceptance Criteria:**
- [x] "저장" 버튼 탭 시 보드가 PNG 파일로 다운로드된다
- [x] 파일명은 `vision-board.png` 형식이다
- [x] 모바일 Safari/Chrome에서 동작한다

---

### US-07 공유
> **As a** 비전보드를 주변에 보여주고 싶은 사용자,
> **I want to** OS 공유 시트를 통해 이미지를 공유하고 싶다,
> **So that** 카카오톡·문자 등으로 바로 전달할 수 있다.

**Acceptance Criteria:**
- [x] "공유" 버튼 탭 시 Web Share API (navigator.share)가 호출된다
- [x] 미지원 브라우저에서는 PNG 다운로드로 폴백된다

---

### US-08 다국어 지원
> **As a** 한국어 외 언어를 사용하는 사용자,
> **I want to** 내 기기 언어로 서비스를 이용하고 싶다,
> **So that** 언어 장벽 없이 비전보드를 만들 수 있다.

**Acceptance Criteria:**
- [x] 한국어 · English · 日本語 · 中文 4개 언어를 지원한다
- [x] 앱 진입 시 기기 언어 설정을 자동 감지하여 언어를 설정한다 (`navigator.language`)
- [x] 헤더의 언어 선택기로 언어를 수동 전환할 수 있다
- [x] UI 텍스트, 추천 키워드가 선택 언어로 표시된다
- [x] 입력된 키워드는 Unsplash 검색을 위해 영어로 자동 변환된다

## 4. 비기능 요구사항

| 구분 | 요구사항 |
|------|---------|
| 성능 | 이미지 로드 포함 보드 생성 5초 이내 |
| 캐싱 | 동일 키워드 이미지·번역 결과를 localStorage에 24시간 캐싱 |
| 접근성 | 터치 타겟 최소 44×44px |
| 반응형 | 모바일 우선 (360px~), 최대 너비 448px |
| 브라우저 | iOS Safari 16+, Android Chrome 110+ |
| 스크롤 | iOS 모멘텀 스크롤 바운스 방지 (`overscroll-behavior-y: none`) |
| ID 생성 | HTTP 환경(내부망 테스트)에서도 동작하는 ID 생성 (`crypto.randomUUID` 폴백) |

- 드래그&드롭 수동 편집

- Unsplash Production 승인 전 50 req/시간 초과 시 자동 대응
